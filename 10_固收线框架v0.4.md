## 0. 设计原则(债基特有,承研报共识)

1. **票息收入不计入能力分,只作标签**(天风/华宝铁律):债基收益主体是票息(NAFMII 实证套息 2-3% 远大于波段),但票息是"持有就有"的 beta,不是经理 alpha。能力评价只看择券/择时/杠杆运用产生的超额。
2. **净值法 + 持仓法双轨**:净值法(华泰/华宝 Campisi 五因子回归)无需持仓、周频可跑、公私募通用,作主算法;持仓法(NAFMII 利润表四能力)作策略标签与排雷,半年频。
3. **纯债 vs 含权分流**:净值五因子模型对纯债 R²>0.7 可信,对含权(二级债基/固收+)R²<0.5 须辅以持仓——这决定了固收+必须单独记分卡。
4. **目标是绝对收益体验**:固收/固收+ 投资者要的是"稳",评价核心是回撤控制与达标率,不是弹性(承映射文档)。

## 1. L0 分类(债基同类组)

主干(Wind 二级 + 实际持仓校验,防漂移):

- **短期纯债**(组合久期短、限投资范围)
- **中长期纯债**
- **混合债券一级**(可打新/转债,不可二级买股)
- **混合债券二级**(≤20% 权益)
- **偏债混合 / 固收+**(权益中枢 10-30%,按中枢再分保守/稳健/积极)
- 可转债基金(单列,权益属性强)

策略标签(正交):利率债型 / 信用债型 / 信用下沉型 / 杠杆套息型 / 摊余成本法 / 持有期。

## 3. L2 记分卡——纯债

**评价目标:回撤控制 + 信用风险甄别 + 可持续的择券 alpha**

### A. 收益质量(权重 25%,低于权益线——纯债收益差异小)

| 指标 | 算法 | 来源 |
|---|---|---|
| A1 净值法 alpha | 华宝五因子周频回归残差(久期中性正交),取 net alpha | 自算(净值+中债指数) |
| A2 持续性(CPR) | NAFMII 胜率持续法:CPR=WW×LL/(WL×LW),要求 >1、P(WW)>0.5、无连续两期垫底 | 自算(滚动分位) |
| A3 同类超额 | 相对同类中位数的年化超额 | 自算 |

### B. 风险控制(权重 35%,债基最重)

| 指标 | 算法 |
|---|---|
| B1 最大回撤 + 修复天数 | 净值序列 |
| B2 卡玛比率 | 年化/最大回撤(核心) |
| B3 下行惩罚 | 晨星 MRAR(0)−MRAR(2) 思路,先用索提诺近似 |
| B4 净值异动排雷 | 偏离久期×利率理论值 → 标记踩雷/大额申赎 |
| B5 月度正收益占比 | 绝对收益体验 |

### C. 收益来源与信用(权重 25%)

| 指标 | 算法 | 来源 |
|---|---|---|
| C1 Campisi 四分 | 票息/国债/利差/择券,择券占超额比(NAFMII:基金间差异 60-70% 来自择券) | 净值五因子 or 且慢债基工具 |
| C2 信用下沉度 | 中债隐含评级打分偏离率(适度为佳,U型) | 且慢 getBondFundCreditRatingLevel |
| C3 久期偏离 | 久期中枢 vs 同类、久期择时有效性 | 且慢 getBondAllocationByFundCode |
| C4 杠杆套息 | NAFMII 杠杆贡献(纯利润表可算) | 持仓 |

### D. 经理与平台(权重 10%)
经验年限、固收团队规模、信用研究支持(沿用主动权益 D 维数据)。

### E. 运作费用(权重 5%)
综合费率(债基费率敏感度高,但差异小)、机构占比。

## 4. L2 记分卡——固收+(差异点)

- **先按权益中枢分组**(保守<10% / 稳健10-20% / 积极20-30%)再比较
- 核心指标:目标回撤达标率(如<3%)、收益回撤比、**股债贡献分解**、权益择时稳定性、转债仓位与风格
- 债底部分沿用纯债 C 维(信用/久期/杠杆)
- 净值五因子 R² 低,**必须辅以持仓归因**(纯净值不可信)

## 6. 净值五因子模型(华宝口径,v0.4 核心算法)

债基周频收益 ~ 五因子(均久期中性正交):
- **level**(久期):中债国债总财富指数
- **slope**(期限结构):中债中短期 − 中债长期(久期中性)
- **credit**(信用利差):中债企业债AAA − 中债国开债(久期中性)
- **default**(违约):中债高收益企业债 − 中债企业债AAA(久期中性)
- **convertible**(可转债):中债转债 − 中债国债(等比例多空)

残差 = 基金 alpha。R²>0.7(纯债)→ alpha 可信;<0.5(含权)→ 标低置信度、辅持仓。
实现:新建 `campisi.py`,周频净值 + 中债指数,约束回归(复用 rbsa.py 的投影梯度求解器)。

## 8. 待定与风险

- NAFMII"套息为王"结论是 2019-21 特定周期产物,权重需回测校准(承核心原则)
- 净值五因子对含权债基失效,固收+ 强依赖持仓(低频)
- 信用下沉的"适度"边界需结合市场信用环境动态调整
- 摊余成本法债基净值平滑,回撤指标失真,需特殊处理
