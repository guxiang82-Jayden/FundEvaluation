# BOND_INDEX 中债系 TE 自检

日期: 2026-06-17

## Step 0 净值源结论

本机可通过 `akshare.bond_index_general_cbond(index_category, indicator="财富", period=...)`
取得中债官网指数族的历史日度财富指数值。接口可按“指数族中文名 + 期限段”取值, 例如:

- `国开行债券总指数 / 1-3年 / 3-5年 / 7-10年 / 总值`
- `农发行债券总指数 / 1-3年 / 3-5年 / 总值`
- `进出口行债券总指数 / 1-3年 / 3-5年 / 5-7年 / 7-10年 / 总值`
- `国债总指数 / 1-3年 / 3-5年 / 7-10年 / 总值`

本机未获得官方 `CBAxxxxx` 代码表。为避免伪造代码, `index_code` 使用透明内部源键
`CBOND_<中债指数族>_<期限段>`, 并在 `data/index_code_cbond_ref.csv` 中记录实际
`return_source` 与 `verified`。
