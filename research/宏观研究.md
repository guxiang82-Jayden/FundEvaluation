| 维度               | 核心问题               | 代表指标                          |
| ---------------- | ------------------ | ----------------------------- |
| **1. 增长与景气**     | 经济是在复苏、过热、放缓还是衰退？  | PMI、工业增加值、社融需求、地产销售、企业盈利预期    |
| **2. 通胀与价格**     | 通胀方向是否制约政策和利率？     | CPI、PPI、核心通胀、大宗商品、工资和租金       |
| **3. 政策与无风险利率**  | 财政和货币政策在宽松还是收紧？    | 政策利率、国债曲线、实际利率、财政赤字、政府支出      |
| **4. 流动性与信用**    | 市场资金是否充裕，融资是否顺畅？   | 社融/M2、DR007、信用利差、信贷脉冲、美元流动性   |
| **5. 估值与风险补偿**   | 当前价格是否透支基本面？       | PE/PB历史分位、股债性价比、ERP、期限利差、信用利差 |
| **6. 风险偏好与市场结构** | 资金愿不愿意承担风险，交易是否拥挤？ | 成交额、波动率、杠杆、ETF资金流、融资余额、市场宽度   |
可以将它们概括为：

> **增长、通胀、政策、流动性、估值、情绪**

其中要特别区分：

- **流动性**：有没有钱，以及融资成本高不高。
- **风险偏好**：资金是否愿意买风险资产。
- **估值**：资产价格是否已经反映预期。
- **增长与盈利**：最终能否兑现价格所隐含的预期。

每个维度：-2 / -1 / 0 / +1 / +2 总分 = Σ（维度得分 × 权重）

建议基础权重：

- 增长与景气：25%
- 政策与利率：20%
- 流动性与信用：20%
- 通胀：10%
- 估值：15%
- 风险偏好：10%

但**不要只看总分**。同样的总分可能对应完全不同的市场状态，例如“增长弱、流动性强”通常利好债券和成长风格，而“增长强、通胀高”更偏向资源、价值或通胀受益资产。
