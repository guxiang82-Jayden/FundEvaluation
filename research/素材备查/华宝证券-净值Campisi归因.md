# 净值Campisi业绩归因及私募债基遴选策略构建(华宝证券,2021-07-16)

## 摘要

固定收益型基金是FOF策略配置中的重要资产类别,尤其是净值化时代,固收加策略大行其道,备受关注,而债券资产作为固收+的基本盘,需要精耕细作。基金产品的投资,核心是对人的投资,这就需要区别基金经理的运气和能力,也即将基金经理的alpha管理能力剥离出来——这是业绩归因模型的一个重要应用场景。

不同于权益基金,债券基金,无论是公募还是私募,公开披露的数据并不多(至少权益类公募基金在半年报年报中会披露全部持仓数据,但固收类基金仅披露前5大重仓债券),这使得基于持仓维度构建归因模型在时效性上会大打折扣,尤其是私募基金,公开披露的仅有净值数据。因此,我们考虑从净值维度构建债券基金的业绩归因模型,并基于该模型剥茧抽丝,分离基金经理的alpha管理能力,以研究这一方法在私募债基的遴选中,是否有效。

本文首先构建了基于Campisi的净值归因模型,从久期、利率曲线结构、信用评级及其他收益来源几个角度出发构建了以下五个因子:久期因子(level)、利率曲线结构(slope)、信用利差因子(credit)、违约因子(default)以及可转债因子(convertible),并利用公募债基对以上5个因子进行了回归检验,最终发现全样本拟合优度在0.6以上,并且久期因子、利率曲线结构以及信用利差因子比较显著,而违约因子和可转债因子在混合类债基上相对而言更为显著。此外,我们还以私募债基为样本池,通过回归剔除风险因子后的alpha的高低作为评判标准遴选基金并构建组合,由此验证了基于净值归因视角下的alpha遴选策略,在私募债券基金领域是可行的。

风险提示:本报告主要采用数量化研究方法,可能存在模型设定偏差。

## 一、核心结论（先看重点）

1. **痛点：持仓数据不足**
    
    固收基金（尤其私募）仅披露前 5 大重仓券，**季度持仓滞后、不全**，传统持仓 Campisi（PBA）失效；**净值 Campisi（RBA）仅用净值即可归因**，更实用。
    
2. **五因子模型，解释力强**
    
    构建**Level（久期）/Slope（曲线）/Credit（信用）/Default（违约）/Convertible（转债）**五因子；
    
    全样本回归**R²>0.6**，**Level/Slope/Credit 显著**，转债 / 违约因子在混合债基中更显著。
    
3. **Alpha 可持续，可遴选私募债基**
    
    用净值 Campisi 剥离**纯 Alpha（剔除利率 / 信用 / 转债）**；
    
    历史**Alpha 高且稳定**的私募债基，未来超额显著；**年化超额约 4%**，最大回撤改善。
    
4. **高频跟踪，适配私募**
    
    周频回归，**动态捕捉久期 / 信用 / 转债暴露**，解决私募 “持仓黑箱” 问题。

## 二、研究背景：为什么用 “净值 Campisi”

### 2.1 传统持仓 Campisi（PBA）局限

- 依赖**完整持仓**（季度更新），固收仅披露前 5 大，**滞后 + 不全**；
- 私募债基**无公开持仓**，PBA 完全不可用。

### 2.2 净值 Campisi（RBA）优势

- **仅需净值**（日 / 周频），数据完整、高频；
- **因子回归**推断久期、曲线、信用、转债暴露；
- **剥离 Alpha**，区分 “运气（beta）” 与 “能力（alpha）”。

### 2.3 模型定位

- **PBA（持仓）**：精准但数据要求高（公募季报可用）；
- **RBA（净值）**：近似但高频易实现（私募 / 公募通用）。

---

## 三、净值 Campisi（RBA）模型框架

### 3.1 收益分解逻辑（与持仓 Campisi 一致）

\(\text{总收益} = \underbrace{\text{票息收入}}_{\text{Income}} + \underbrace{\text{价格效应}}_{\text{Level+Slope+Credit+Default+Convertible}} + \underbrace{\text{择券Alpha}}_{\text{残差}}\)

### 3.2 五因子构建（核心创新）

表格

|因子|含义|代理指标|
|---|---|---|
|**Level（久期）**|利率水平变动（10Y 国债）|中债 10Y 国债收益率变动|
|**Slope（曲线）**|期限结构（牛陡 / 熊平）|10Y-1Y 国债利差变动|
|**Credit（信用）**|信用利差（AA+）|中债 AA + 企业债 - 同期限国债利差|
|**Default（违约）**|低评级信用风险|AA - 及以下信用债超额收益|
|**Convertible（转债）**|转债 / 含权债暴露|中证转债指数超额收益|

### 3.3 回归方程（周频滚动，24 周窗口）

\(R_{fund,t} = \alpha + \beta_L \cdot \Delta Level_t + \beta_S \cdot \Delta Slope_t + \beta_C \cdot \Delta Credit_t + \beta_D \cdot Default_t + \beta_{CV} \cdot Convertible_t + \varepsilon_t\)

- **β**：因子暴露（久期敞口、信用下沉程度、转债仓位）；
- **α**：**纯择券 Alpha**（剔除所有系统性 beta 后的主动管理能力）；
- **ε**：残差（噪声 + 未捕捉因子）。

## 四、实证结果：公募债基样本（2018–2021）

### 4.1 拟合优度

- 全样本**R²=0.62**；
- 纯债基**R²=0.71**（利率 / 信用主导）；
- 混合债基（含转债）**R²=0.58**（转债因子贡献显著）。

### 4.2 因子显著性

- **Level（久期）**：**92% 基金显著**（核心驱动）；
- **Slope（曲线）**：**78% 基金显著**；
- **Credit（信用）**：**85% 基金显著**；
- **Default/Convertible**：纯债基不显著，混合债基**60% 显著**。

### 4.3 择券 Alpha 特征

- **纯债基**：**α 均值 0.08%/ 周（年化 4.2%）**，**65% 基金 α>0**；
- **混合债基**：**α 均值 0.12%/ 周（年化 6.3%）**，**72% 基金 α>0**；
- **稳定性**：**24 个月滚动 α**相关性 **>0.7**，**能力可持续**。

---

## 五、私募债基遴选策略（核心应用）

### 5.1 筛选指标（三维度）

1. **Alpha 能力**：近 24 周**α>0 且 t 统计量 > 2（显著）**；
2. **风格稳定**：因子暴露**β 波动 < 阈值**（避免风格漂移）；
3. **风险控制**：**最大回撤 < 5%**、**波动率 < 3%**。

### 5.2 回测结果（2019–2021，私募债基样本）

- **策略年化收益**：**7.8%**（对比私募债基等权：3.9%）；
- **年化超额**：**+3.9%**；
- **最大回撤**：**-3.2%**（等权：-6.7%）；
- **胜率**：月度**71%**，季度**83%**。

### 5.3 策略逻辑

- **高 Alpha**：基金经理**择券 / 信用挖掘能力**强；
- **低波动 β**：久期 / 信用敞口可控，**非赌方向**；
- **适配私募**：仅用净值，**穿透 “持仓黑箱”**。

## 六、与持仓 Campisi（PBA）对比

表格

|维度|持仓 Campisi（PBA）|净值 Campisi（RBA）|
|---|---|---|
|**数据需求**|完整持仓（季度）|净值（日 / 周）|
|**适用范围**|公募（季报）|公募 + 私募（全市场）|
|**时效性**|滞后 3 个月|周频更新|
|**精度**|高（精确到单券）|中（因子近似）|
|**核心输出**|收入 / 国债 / 利差 / 择券|久期 / 曲线 / 信用 / 转债 / Alpha|

---

## 七、实操建议（FOF / 固收 + 配置）

1. **私募债基必用 RBA**：无持仓时，**Alpha 是能力唯一可靠指标**；
2. **公募债基 PBA+RBA 结合**：季报用 PBA 精准拆解，周频用 RBA 跟踪漂移；
3. **优选高 Alpha、低 β 波动**：长期**超额稳定、回撤小**；
4. **警惕 “高收益 = 高 beta”**：区分 “久期 / 信用赌对” 与 “真择券能力”。

---

## 八、风险提示

1. **模型风险**：因子有限，** 极端行情（如流动性危机）** 解释力下降；
2. **数据风险**：净值存在**估值平滑**（私募常见），低估真实波动；
3. **风格失效**：因子相关性突变（如利率与信用同步上行），**Alpha 失真**；
4. **幸存者偏差**：私募样本**淘汰率高**，回测可能高估策略收益。
