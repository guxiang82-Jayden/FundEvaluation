## 一、核心结论（先看重点）

1. **财报归因是公募债基最优解**：公募仅披露前 5 大重仓券，**持仓 Campisi 不可用**；财报归因**无需持仓、数据准确**，适配全公募债基。
2. **收益拆成三大块**：
    
    - **债券收益**：杠杆收益 + 非杠杆（票息 + 资本利得（交易 + 公允价值））
    - **股票收益（含权债基）**：股利 + 资本利得
    - **其他收益**：ABS、存款、逆回购、正回购成本、费用。
    
3. **纯债基效果最好**：可转债收益无法从财报中拆分，**纯债基归因精度 > 含权债基**。
4. **两类典型基金画像**：
    
    - **票息型**：短久期、中评级、高票息、低交易
    - **交易型**：久期波动大、低票息、高资本利得。

## 二、模型背景：为什么用财报归因

### 2.1 传统持仓 Campisi 的痛点

- 依赖**完整持仓（季度）**，公募仅披露前 5 大，**滞后 + 不全**。
- 私募债基**无公开持仓**，Campisi 完全不可用。

### 2.2 财报归因的优势

- **无需持仓**：仅用**半年报 / 年报利润表 + 附注**。
- **数据准确**：会计准则统一，**可横向 / 纵向对比**。
- **覆盖广**：纯债、一级、二级债基**全适配**。

### 2.3 财报归因的局限

- **频率低**：半年 / 年报，**滞后 2–3 个月**。
- **可转债混在一起**：无法区分普通债与转债收益，**含权债基精度下降**。

## 三、财报归因模型框架（核心）

### 3.1 总收益拆分逻辑

\(\text{基金总利润} = \underbrace{\text{债券收益}}_{\text{杠杆+非杠杆（票息+资本利得）}} + \underbrace{\text{股票收益（含权）}}_{\text{股利+资本利得}} + \underbrace{\text{其他收益}}_{\text{ABS+存款+逆回购-正回购成本-费用}}\)

### 3.2 关键指标与对应会计科目（2022 年新规后）

- **债券票息**：取自**债券投资收益项目构成表→利息收入**（2022 中报后，利息不在利润表 “利息收入”）。
- **债券交易收益**：债券投资收益 − 债券利息收入。
- **债券公允价值变动**：附注→公允价值变动收益→债券。
- **杠杆收益**：(债券投资收益 + 债券公允价值变动) × (平均债券仓位−100%)/100%。
- **杠杆超额收益**：杠杆收益 − 正回购成本。

### 3.3 归一化处理（横向对比用）

- **非杠杆票息 / 资本利得**：按 ** 债券仓位 100%** 归一化，消除仓位差异影响。
- **平均资产净值**：净利润 / 加权平均净值利润率（年报 / 半年报披露）。

---

## 四、财报变化：2022 年 5 月新规影响

- **旧规（2022 中报前）**：债券利息在利润表 “利息收入” 单独列示。
- **新规（2022 中报后）**：
    
    - 摊余成本计量的银行存款 / 逆回购：仍在 “利息收入”。
    - 债券 / ABS 利息：移入**投资收益**，需从**附表**提取。
    
- **实操要点**：2022 中报后必须查**债券投资收益项目构成表**拿票息。

## 五、实证案例：两类纯债基金对比

### 5.1 基金 A：票息主导型（短久期、中评级）

- **收益结构**：债券收益占 90%+，**票息是核心（60%+）**，资本利得低。
- **持仓特征**：久期≈1 年，重仓 AA/AA - 信用债，**信用挖掘增厚票息**。
- **杠杆**：2022 年前无杠杆，2022 年后小幅正回购，**杠杆超额收益为正**。

### 5.2 基金 B：交易主导型（久期波动大、高等级）

- **收益结构**：债券收益占 90%+，**资本利得主导（50%+）**，票息偏低。
- **持仓特征**：久期 1–4 年大幅波动，**利率债波段 + 高等级信用底仓**。
- **杠杆**：仓位≈100%，**正回购成本高**，杠杆超额收益为负。

### 5.3 关键差异对比

表格

|维度|基金 A（票息型）|基金 B（交易型）|
|---|---|---|
|票息收益|高（同类前 20%）|低（同类后 30%）|
|资本利得|低（公允价值为主）|高（交易为主）|
|久期|短（≈1 年）、稳定|长（1–4 年）、波动大|
|信用评级|中（AA/AA-）|高（AAA / 利率债）|
|杠杆收益|小幅正贡献|几乎为 0|

## 六、实操步骤（可直接落地）

1. **取数**：下载基金**年报 / 半年报**，提取：
    
    - 利润表：利息收入、投资收益、公允价值变动、费用。
    - 附注：债券 / 股票 / ABS 公允价值变动明细。
    - 附表：债券投资收益项目构成→利息收入。
    
2. **计算**：
    
    - 债券票息 = 附表利息收入。
    - 债券交易收益 = 债券投资收益 − 票息。
    - 债券资本利得 = 交易收益 + 公允价值变动。
    - 杠杆收益 = (投资收益 + 公允价值变动) × (仓位−100%)/100%。
    
3. **归一化**：票息 / 资本利得按**100% 债券仓位**调整。
4. **归因结论**：判断基金是**票息型、交易型、杠杆型**或混合。

## 八、风险提示

1. **数据滞后**：半年 / 年报披露晚，**无法及时跟踪风格漂移**。
2. **可转债干扰**：含权债基中**转债收益无法拆分**，归因结果偏误。
3. **会计口径差异**：少数基金附表披露不完整，**需手工校验**。
4. **不构成投资建议**：仅为历史归因，**不代表未来收益**。
