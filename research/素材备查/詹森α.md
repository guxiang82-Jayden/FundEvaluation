# 一、Jensen's Alpha（詹森 α）完整解读

## 1. 核心公式

基于**CAPM 资本资产定价模型**推导：

Rp - Rf = \alpha_p + \beta_p(R_m - R_f) + \varepsilon\)

变形得到詹森 α：

\(\boldsymbol{\alpha_p = (R_p-R_f) - \beta_p(R_m-R_f)}\)

### 变量释义

- \(R_p\)：投资组合实际收益率
- \(R_f\)：无风险收益率
- \(R_m\)：市场基准组合收益率
- \(\beta_p\)：组合贝塔（系统性风险）
- \(\alpha_p\)：**詹森阿尔法**（回归方程截距项）

## 2. 指标含义 & 主动管理价值判断

詹森 α 本质是：**扣除「风险匹配后市场应得收益」，剩余的纯主动收益**，专门用来评判**基金 / 管理人的主动管理能力**。

1. \(\boldsymbol{\alpha > 0}\)：组合实际收益 > 同风险下市场理论收益
    
    主动管理创造了**正向超额收益**，管理人具备选股 / 择时能力。
2. \(\boldsymbol{\alpha = 0}\)：收益完全匹配 CAPM 理论值
    
    主动管理没有额外价值，收益仅来自承担市场风险。
3. \(\boldsymbol{\alpha < 0}\)：组合跑输同风险基准
    
    主动操作反而拖累收益，不如被动持有市场指数。

### 核心意义（对应你原文）

1968 年詹森提出该指标后，正式拉开**主动管理能否持续创造价值**的学术与行业辩论：

- 支持主动管理：长期能跑出稳定正 α，证明基金经理选股、择时、行业轮动有效；
- 质疑主动管理：绝大多数基金 α 长期趋近于 0 甚至为负，主动管理无法战胜市场，不如被动指数。

同时它和特雷诺比率形成统一逻辑：**资产评价必须做风险调整**—— 不能只看绝对收益，高收益往往对应高风险，只有把风险剔除后的收益，才具备对比意义，这也成为行业共识。

# 二、Roll (1977) 罗尔批判（核心争议点）

## 1. 核心论点：**市场组合不可观测**

CAPM 与詹森 α 的成立，有一个前置假设：公式里的\(R_m\)是**真正的全市场组合**。

理论上的市场组合：包含**全球所有风险资产**（股票、债券、大宗商品、不动产、另类资产等全部可投资标的），是一个理论抽象概念。

现实中存在两大问题：

1. **无法构建真实市场组合**
    
    实操里所有人都只能用**替代基准**：沪深 300、中证全指、标普 500 等股票指数，仅代表单一品类资产，并不是 CAPM 定义的完整市场组合。
2. **基准选择直接扭曲评价结果**
    
    詹森 α、贝塔、特雷诺比率全都依赖基准\(R_m\)，**换一个基准，α 的正负、大小会彻底改变**。

## 2. 举例直观理解

假设一只偏债混合型基金：

- 用**纯股票指数**当基准：基金 β 偏低，算出 α 大概率为负，被判定为 “主动管理失效”；
- 改用**股债综合指数**当基准：风险匹配更合理，α 可能转正，评价结论完全反转。

这就是 Roll 批判的核心：

> 基于 CAPM 的基金绩效评价，**本质是 “基准选择的游戏”**，而非纯粹对投资能力的客观评判。只要基准选得不一样，绩效排名、能力判断都会失真。

## 3. 延伸影响

1. 动摇了詹森 α、特雷诺比率等传统风险调整指标的**绝对客观性**；
2. 后续行业演化出多因子模型（Fama-French 三因子 / 五因子）、风格基准、定制化基准，用来缓解单一市场基准的缺陷；
3. 至今仍是资产绩效评价领域的经典争议：**没有完美基准，所有风险调整收益都存在基准偏差**。

# 三、三者极简对比（Treynor / Jensen α / Roll 批判）

1. **Treynor**：单位系统性风险的超额收益，侧重**风险收益效率**；
2. **Jensen α**：CAPM 回归截距，直接衡量**主动管理超额收益**，确立「风险调整收益」行业共识；
3. **Roll 批判**：指出底层漏洞 —— 理论市场组合不存在，**基准选择决定评价结果**，传统指标并非绝对客观。
