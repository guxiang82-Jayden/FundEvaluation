## 一、核心结论（先看重点）

1. **双框架互补**：长江金工构建 ** 多因子归因（上）+ 持仓归因（下）** 双体系，覆盖 “风险暴露 + 主动决策” 全维度。
2. **四大持仓模型**：系统梳理**Brinson、Wagner-Tito、加权久期、Campisi**，**Campisi 最适配债基**（利率 + 信用 + 择券三维度）。
3. **Campisi 本土化**：针对国内债基**持仓不全（仅前 5 大）**问题，提出**重仓券加权 + 基金久期校准**的改进方案。
4. **因子 + 持仓结合**：推荐**多因子（高频跟踪）+Campisi（季度深度）**交叉验证，精准识别**久期择时、信用下沉、择券能力**。

---

## 二、研究背景：债基归因的特殊性

### 2.1 权益模型不适用

- Brinson 侧重**行业配置 + 个股选择**，忽略债基核心的**久期 / 利率风险**；
- 债基收益驱动：**票息（60%-80%）+ 利率波动 + 信用利差 + 择券**，与权益完全不同。

### 2.2 国内数据痛点

- 公募债基**仅披露前 5 大重仓券**（占比 30%-50%），**完整持仓缺失**；
- 私募债基**无公开持仓**，传统持仓归因受限；
- 解决方案：**因子归因（净值）+ 改进 Campisi（重仓）**。

### 2.3 长江归因体系定位

- **上篇（多因子）**：基于净值，**高频（周 / 月）**，跟踪**久期、曲线、信用、转债**暴露；
- **下篇（持仓）**：基于季报，**低频（季度）**，拆解**配置、择时、择券**贡献。

## 三、上篇：债基多因子分析体系（净值归因）

### 3.1 核心逻辑

- 用**净值回归**替代持仓，**每日 / 每周**更新因子暴露；
- 剥离**系统性风险（Beta）**，提取**纯 Alpha（主动管理能力）**。

### 3.2 五因子模型（长江原创）

表格

|因子|含义|代理指标|
|---|---|---|
|**Level（久期）**|利率水平变动|中债 10Y 国债收益率|
|**Slope（曲线）**|期限结构（牛陡 / 熊平）|10Y-1Y 国债利差|
|**Credit（信用）**|信用利差（AAA/AA+）|中债 AA + 企业债 - 国债利差|
|**Default（违约）**|低评级信用风险|AA - 及以下信用债超额收益|
|**Convertible（转债）**|转债 / 含权债暴露|中证转债指数超额收益|

### 3.3 回归方程与输出

\(R_{fund,t} = \alpha + \beta_L Level_t + \beta_S Slope_t + \beta_C Credit_t + \beta_D Default_t + \beta_{CV} Convertible_t + \varepsilon_t\)

- **β**：因子暴露（久期敞口、信用下沉程度、转债仓位）；
- **α**：**纯择券 Alpha**（剔除利率 / 信用 / 转债后的主动收益）；
- **R²**：模型解释力（纯债基 > 0.7，含权债基 > 0.5）。

### 3.4 应用场景

- **风格监控**：跟踪**久期漂移、信用下沉、转债仓位**变化；
- **能力筛选**：优选**α>0 且 t 统计量 > 2**、**β 稳定**的基金；
- **私募适配**：无持仓时，**Alpha 是能力唯一可靠指标**。

## 四、下篇：基于持仓的债基归因模型（核心）

### 4.1 模型梳理：四大主流方法

#### 4.1.1 Brinson 分解模型（权益移植）

- 超额收益 =**券种配置效应 + 个券选择效应 + 交互效应**；
- 缺陷：**无久期维度**，对债基针对性弱，仅适用于**大类配置对比**。

#### 4.1.2 Wagner-Tito 分解模型

- 收益 =**票息收益 + 利率波动收益 + 信用利差收益 + 择券收益**；
- 特点：**区分利率与信用**，但**久期计算复杂**，实操少。

#### 4.1.3 加权久期分析方法

- 核心：**久期 × 利率变动**= 利率贡献；
- 缺陷：**忽略信用利差与择券**，仅适用于**纯利率债基**。

#### 4.1.4 Campisi 分解模型（推荐）

- **收益 = 收入效应（票息）+ 国债效应（利率）+ 利差效应（信用）+ 择券效应（Alpha）**；
- 优势：**覆盖债基全收益来源**，逻辑清晰、实操性强。

### 4.2 Campisi 模型详解（长江改进版）

#### 4.2.1 原始模型（完整持仓）

1. **收入效应（Income）**：持有期票息收益（稳定核心）；
2. **国债效应（Treasury）**：无风险利率变动导致的价格波动（久期驱动）；
3. **利差效应（Spread）**：信用利差变动导致的价格波动（信用风险）；
4. **择券效应（Selection）**：残差，**基金经理主动择券能力**。

#### 4.2.2 长江改进（适配国内持仓不全）

- **重仓券加权**：用**前 5 大重仓券**替代全持仓，按**市值加权**计算因子暴露；
- **久期校准**：用基金披露的**组合久期**修正重仓券久期偏差；
- **基准匹配**：按基金类型（纯债 / 一级 / 二级）匹配**中债综合 / 信用债 / 转债指数**基准。

### 4.3 实证案例：纯债基归因（2018Q3）

- **收入效应**：**1.2%**（占比 70%，票息主导）；
- **国债效应**：**-0.3%**（利率上行，负贡献）；
- **利差效应**：**0.4%**（信用利差收窄，正贡献）；
- **择券效应**：**0.5%**（基金经理择券能力强）；
- **总收益**：**1.8%**（与净值一致）。

### 4.4 模型对比（核心差异）

表格

|维度|Brinson|Wagner-Tito|加权久期|Campisi（长江）|
|---|---|---|---|---|
|**票息**|✅|✅|✅|✅（核心）|
|**利率（久期）**|❌|✅|✅（唯一）|✅|
|**信用利差**|❌|✅|❌|✅|
|**择券能力**|✅|✅|❌|✅（残差）|
|**实操难度**|低|高|低|中（改进后易）|
|**国内适配**|差|中|中|**优（重仓 + 久期）**|

## 五、多因子 vs Campisi：互补与结合

### 5.1 核心区别

表格

|维度|多因子归因（净值）|Campisi 归因（持仓）|
|---|---|---|
|**数据**|净值（日 / 周）|季报持仓（季度）|
|**频率**|高频（跟踪漂移）|低频（深度拆解）|
|**输出**|因子暴露（β）+Alpha|票息 + 利率 + 信用 + 择券|
|**精度**|中（因子近似）|高（持仓驱动）|
|**适用**|全市场（含私募）|公募（季报）|

### 5.2 结合策略（推荐实操）

1. **高频监控（多因子）**：周频回归，跟踪**久期、信用、转债**暴露，及时发现**风格漂移**；
2. **季度深度（Campisi）**：季报后用改进 Campisi 拆解**票息、利率、信用、择券**贡献，评估**真实能力**；
3. **交叉验证**：**多因子 Alpha**与**Campisi 择券效应**相关性 > 0.7，结果可信；
4. **基金优选**：同时满足**高 Alpha、稳定因子暴露、正择券效应**。

## 六、实操步骤（改进 Campisi，可直接落地）

1. **数据获取**：
    
    - 季报：前 5 大重仓券（代码、市值、久期、评级、票息）；
    - 基金：组合久期、净值、类型（纯债 / 一级 / 二级）；
    - 基准：对应指数（中债综合、信用债、转债）收益率。
    
2. **计算各效应**：
    
    - **收入效应**：Σ（单券票息 × 权重）；
    - **国债效应**：- 组合久期 × 国债收益率变动；
    - **利差效应**：Σ（单券利差 × 权重）变动；
    - **择券效应**：总收益 - 收入 - 国债 - 利差。
    
3. **结果解读**：
    
    - 票息高：**票息型**（短久期、中评级）；
    - 国债效应大：**久期择时型**（波段操作）；
    - 利差效应大：**信用下沉型**（高收益债）；
    - 择券效应高：**精选个券型**（低波动、稳超额）。

## 七、风险提示

1. **数据局限**：重仓券≠全持仓，**小权重券收益无法捕捉**，存在偏差；
2. **模型假设**：因子有限，** 极端行情（流动性危机、信用违约潮）** 解释力下降；
3. **基准选择**：基准不匹配会导致**效应分解失真**，需严格按基金类型匹配；
4. **幸存者偏差**：样本**淘汰率高**，回测可能高估策略收益。
